本目录包含 XiaoBa 学术写作工具的使用示例和最佳实践。

## 快速开始

### 1. 搜索文献
```
用户: 帮我搜索关于[主题]的最新论文
```

### 2. 分析论文质量
```
用户: 帮我分析一下我的论文质量
```

### 3. 生成引用
```
用户: 帮我生成这篇论文的APA格式引用
```

## 工作流程

XiaoBa 推荐的论文写作流程：

1. **文献调研** → 使用 `search_papers` 和 `fetch_paper`
2. **大纲设计** → 与 XiaoBa 讨论论文结构
3. **内容撰写** → 逐章节完成内容
4. **质量检查** → 使用 `analysis_tool` 检查质量
5. **格式规范** → 使用 `convert_tool` 转换格式
