# XiaoBa CLI

XiaoBa - 智能 AI 对话助手，支持多平台机器人接入。

## 功能特性

- 🖥️ **桌面应用** - Electron 打包，支持 Windows/macOS
- 🎯 **Dashboard** - 可视化管理界面
- 🤖 **多平台支持** - 飞书、Cats Company、微信机器人
- 🔌 **Skill 系统** - 可扩展的能力插件
- 🏪 **Skill 商店** - 从官方 Hub 安装社区 Skills

## 下载安装

### Windows
下载 [XiaoBa Setup 0.1.0.exe](https://github.com/buildsense-ai/XiaoBa-CLI/releases) 并安装。

### macOS
```bash
git clone https://github.com/buildsense-ai/XiaoBa-CLI.git
cd XiaoBa-CLI
npm install
npm run electron:build:mac
```

## 开发

```bash
npm install
npm run electron:dev
npm run electron:build:win
```
