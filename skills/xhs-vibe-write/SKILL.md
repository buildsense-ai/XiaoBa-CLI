# 小红书风格写作

快速创作符合小红书平台特色的笔记内容。

## 何时使用

**必须使用：**
- 用户要求写小红书笔记
- 需要小红书风格的内容
- 创作社交媒体短内容

## 小红书写作特点

### 1. 标题设计（关键！）

**标题公式：**
```
[Emoji] + [痛点/好奇] + [数字/结果] + [Emoji]
```

**示例：**
- ✨ 熬夜党必看！3个方法让你第二天不显老 💕
- 🔥 月薪5k也能存钱！我的极简理财法 💰
- 📚 考研上岸学姐的血泪教训！别再踩这些坑 ⚠️

**标题要点：**
- 使用 1-3 个相关 emoji
- 制造好奇或共鸣
- 包含具体数字或结果
- 控制在 20-30 字

### 2. 内容结构

**标准结构：**
```
1. 开头（引起共鸣）
   - 痛点描述 / 场景代入
   - "姐妹们" / "宝子们" 等亲切称呼

2. 主体（干货内容）
   - 分点列举（3-5 点）
   - 每点配 emoji
   - 简洁实用

3. 结尾（互动引导）
   - 总结要点
   - 引导点赞收藏
   - 话题标签
```

### 3. 语言风格

**DO：**
- ✅ 口语化、接地气
- ✅ 使用"姐妹们"、"宝子们"、"集美们"
- ✅ 大量使用 emoji（但不过度）
- ✅ 短句为主，易读
- ✅ 制造共鸣感

**DON'T：**
- ❌ 过于正式或书面化
- ❌ 长篇大论
- ❌ 没有重点
- ❌ 缺少互动感

### 4. Emoji 使用指南

**常用 Emoji 分类：**

**强调重点：**
✨ 💡 🔥 ⚡ 💯 👍 ✅

**情感表达：**
😭 😍 🥰 😱 🤔 💕 ❤️

**分类标记：**
📌 📍 🎯 💰 📚 🏠 🍜

**警示提醒：**
⚠️ ❗ 🚫 ⛔

**互动引导：**
👇 👉 💬 📝 🔖

## 写作流程

### Step 1: 明确主题和目标

**询问用户：**
- 主题是什么？
- 目标读者是谁？
- 想传达什么价值？

### Step 2: 设计吸睛标题

**使用标题公式：**
- 痛点型：解决读者的困扰
- 好奇型：制造悬念
- 干货型：承诺具体价值
- 对比型：前后对比效果

### Step 3: 构建内容框架

**开头（50字内）：**
- 场景代入或痛点共鸣
- 使用"姐妹们"等亲切称呼

**主体（300-500字）：**
- 3-5 个要点
- 每点配 emoji 和小标题
- 简洁实用，避免废话

**结尾（50字内）：**
- 总结核心价值
- 引导互动（点赞、收藏、评论）
- 添加话题标签

### Step 4: 润色优化

**检查清单：**
- [ ] 标题是否吸引人？
- [ ] 开头是否引起共鸣？
- [ ] 内容是否有干货？
- [ ] Emoji 使用是否恰当？
- [ ] 语气是否亲切自然？
- [ ] 结尾是否引导互动？

## 执行要求

**直接执行，简洁回复：**
1. 使用 write_file 保存生成的笔记
2. 保存路径：`docs/xhs/xhs_[主题]_[日期].md`
3. 如果 docs/xhs 目录不存在，先创建目录
4. 完成后只回复：`已创建小红书笔记：docs/xhs/[文件名]`
5. 不要解释笔记内容、不要列举亮点、不要提供额外建议
