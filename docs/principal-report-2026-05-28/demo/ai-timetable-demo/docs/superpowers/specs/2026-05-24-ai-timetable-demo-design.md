# AI 智能排课 Web 演示版设计

目标是做一个老师能直接理解的三栏式 Web 工作台。老师不需要知道 OR-Tools、CP-SAT 或 JSON，只需要上传或查看数据、输入自然语言规则、查看自动生成后的课表和冲突提示。

## 第一版范围

- 使用内置真实风格模拟数据作为默认数据源。
- 支持 Excel 上传预览，先帮助老师确认“系统能读到文件”，正式映射导入放到后续版本。
- 支持自然语言规则输入，并展示成人话规则卡片。
- 使用 OR-Tools CP-SAT 生成课表。
- 右侧显示班级课表、冲突提示、规则应用结果。

## 页面结构

- 左侧：教师、教室、时间、课程、已有规则。
- 中间：AI Agent 对话区，老师输入自然语言。
- 右侧：富媒体课程表，包括班级切换、时间轴、冲突提示、自动刷新。

## 技术结构

- 前端：原生 HTML、CSS、JavaScript，降低构建复杂度。
- 后端：Python FastAPI。
- 排课引擎：Google OR-Tools CP-SAT。
- Excel 预览：openpyxl。

## 老师视角原则

页面不展示专业算法词。所有按钮和提示使用“自动排课”“发现冲突”“必须满足”“尽量满足”“重新生成”等学校场景语言。
