# 广大附教育智能体服务协议

甲方：________________________

乙方：________________________

签署日期：______年____月____日

## 第一条 合作背景

1. 甲方希望在学校教师办公、部门协同、资料整理、教务支持、总务台账、学校知识库等真实工作场景中引入 AI 能力，提升资料处理、流程辅助和信息检索效率。
2. 乙方具备教育智能体建设、部署、场景配置、资料库整理和使用支持能力，愿意按本协议约定向甲方提供“广大附教育智能体”服务。
3. 本服务以辅助甲方日常教育教学和学校管理工作为目标。服务过程中形成的使用记录、流程模板、场景材料和成果素材，可由甲方用于校内管理、实践总结和工作汇报。

## 第二条 服务内容

乙方根据甲方实际需求，向甲方提供以下服务：

1. 提供云端“广大附教育智能体”访问与使用服务。
2. 根据甲方确认的使用范围，为老师、教务、总务、校办等角色配置相应入口、账号和权限。
3. 建设并维护学校资料库，支持甲方授权资料的归档、检索、问答、引用和模板复用。
4. 围绕甲方真实工作流程配置教育智能体能力，包括但不限于文档材料、会议纪要、资料整理、教务排课辅助、总务盘点与合同台账、学校资料库检索等场景。
5. 为甲方提供必要的使用培训、场景说明、问题反馈处理和持续优化支持。
6. 根据甲方使用反馈，持续优化教育智能体的提示、流程、模板、资料库目录和常用工作能力。

## 第三条 服务期限与甲方停止服务权

1. 本协议自双方签字或盖章之日起生效。双方按季度进行服务结算，每一季度为一个服务周期。
2. 本协议为持续性服务协议。除甲方停止服务或双方另行书面终止外，服务可按季度持续续行。
3. 甲方有权随时无条件停止使用本服务。甲方停止服务时，仅需以书面、邮件、企业微信、微信或双方认可的其他方式通知乙方，无需说明理由，且不承担违约责任。
4. 乙方收到甲方停止服务通知后，应配合甲方完成账号停用、资料导出、数据删除或停止处理等工作。已支付但未实际发生服务的费用，双方按未服务期间折算后退还或抵扣后续费用；双方另有书面约定的，从其约定。
5. 服务停止不影响停止前已发生的费用结算、保密义务、数据安全义务、知识产权归属及本协议中按性质应继续有效的条款。

## 第四条 服务费用与结算

1. 本服务标准费用为人民币 48,000 元/季度。
2. 甲方按季度向乙方支付服务费。具体付款日期、收款账户、发票开具方式由双方在签署本协议时或后续书面确认。
3. 首个服务周期服务费减免安排：
   - □ 不适用。
   - □ 乙方对甲方首个服务周期服务费予以减免。减免周期：自______年____月____日起至______年____月____日止。
4. 服务费减免期间，乙方提供服务的内容、数据安全、保密、成果归属、人工复核、服务停止和其他双方权利义务，仍按本协议约定执行。
5. 服务费减免期届满后，如甲方继续使用本服务，双方按本协议约定的标准费用执行；双方另有书面约定的，从其约定。
6. 如甲方需要新增超出本协议范围的深度定制开发、第三方系统接入、大规模历史资料治理、专属部署或其他专项服务，双方另行确认服务内容、周期和费用。

## 第五条 甲方职责

1. 指定校内项目联系人或管理员，负责与乙方对接账号开通、权限范围、资料接入、场景优先级和反馈事项。
2. 根据服务需要，协调相关部门或老师提供必要的资料、模板、流程说明、示例文件和业务反馈。
3. 对涉及学生、家长、教师、成绩、合同、入学材料等敏感信息的资料，甲方应确认授权范围；必要时先行脱敏或要求乙方按甲方要求进行脱敏处理。
4. 对教育智能体输出的排课建议、成绩分析、合同台账、入学材料整理、通知文案、报告材料等内容进行人工审核和最终确认。
5. 确认哪些资料、截图、案例、反馈和成果可以用于校内汇报、对外展示或其他用途。
6. 按本协议约定支付服务费用。

## 第六条 乙方职责

1. 按本协议约定向甲方提供“广大附教育智能体”服务，并负责必要的部署、配置、优化和基础运维。
2. 在甲方授权范围内处理资料和数据，不超出本协议目的使用甲方资料和数据。
3. 为甲方提供使用培训、问题反馈处理、场景配置和持续优化支持。
4. 按甲方确认的权限范围配置账号、资料访问和角色使用权限。
5. 采取合理技术和管理措施保护甲方资料、账号、日志和相关数据安全。
6. 未经甲方书面同意，不得将甲方资料、校内流程、师生信息、使用截图、老师反馈、学校名称或校徽用于对外宣传、案例展示或其他客户服务。

## 第七条 人工复核与业务边界

1. 双方确认，教育智能体输出内容属于辅助性建议、整理结果或草稿，不替代甲方现有教学、教务、行政、财务、招生、学生管理、合同管理等制度流程。
2. 涉及排课、调课、成绩分析、学生评价、入学材料、合同台账、对外文件等事项，最终结果以甲方相关负责人或老师人工审核确认为准。
3. 乙方不对甲方未经复核直接使用教育智能体输出内容产生的业务后果承担责任，但乙方应对因其故意或重大过失造成的问题承担相应责任。

## 第八条 数据安全与个人信息保护

1. 乙方仅在甲方授权范围内，为提供本协议项下服务之目的处理甲方资料和数据。
2. 未经甲方书面同意，乙方不得将甲方资料和数据用于其他客户项目、公开展示、商业宣传或训练非甲方专属模型。
3. 涉及未成年人个人信息、学生成绩、家庭信息、入学材料、教师个人信息等敏感资料的，双方应遵循合法、正当、必要和最小范围原则处理。
4. 乙方应根据服务需要采取合理安全措施，包括但不限于权限控制、访问限制、必要日志记录、数据隔离和异常问题处理。
5. 乙方使用云服务、大模型、OCR、文档解析、语音识别等第三方能力时，应确保第三方能力仅为履行本协议目的使用，并采取合理措施避免甲方资料和数据被超范围处理。
6. 服务停止或终止后，乙方应按甲方要求协助甲方导出资料，并在合理期限内删除或停止处理甲方数据；法律法规或监管要求另有规定的除外。

## 第九条 账号与权限管理

1. 甲方指定管理员或联系人确认使用人员名单、账号范围、资料接入范围和权限配置规则。
2. 乙方根据甲方确认的信息进行账号配置和权限设置。
3. 甲方使用人员应妥善保管账号和登录凭证，不得将账号提供给无关人员使用。
4. 如发生账号泄露、异常访问或权限变更需求，甲方应及时通知乙方，乙方应配合处理。

## 第十条 成果归属与使用限制

1. 甲方提供的资料、数据、文档、模板、校内流程、校本案例和由甲方确认形成的校本内容归甲方所有。
2. 乙方在服务前已拥有或独立开发的系统、底层代码、通用模型调用能力、通用工具、通用模板和通用方法归乙方所有。
3. 双方在服务过程中形成的校本资料库目录、校本模板、流程配置和应用案例，甲方有权在本校范围内持续使用；其中不包含乙方底层系统、代码和通用技术能力的所有权转移。
4. 未经甲方书面同意，乙方不得向第三方披露或复用含有甲方身份、资料、业务流程或师生信息的专属成果。

## 第十一条 服务支持与维护

1. 乙方通过双方确认的沟通渠道提供服务支持，包括微信、企业微信、电话、邮件或项目群等。
2. 乙方在工作日对甲方反馈的问题进行响应；对影响主要功能使用的紧急问题，乙方应优先处理并及时告知进展。
3. 因系统维护、云服务异常、网络故障、第三方服务异常等原因影响服务时，乙方应及时通知甲方并尽快恢复服务，尽量减少对甲方使用的影响。
4. 乙方可根据服务安全、性能优化或功能升级需要进行必要维护，但应尽量选择对甲方影响较小的时间进行。

## 第十二条 保密条款

1. 双方应对在合作过程中获知的对方非公开信息承担保密义务。
2. 保密信息包括但不限于学校资料、师生信息、业务流程、账号信息、技术方案、价格信息、合同内容、使用数据和未公开成果。
3. 未经对方书面同意，任何一方不得向第三方披露保密信息；法律法规、监管机关、司法机关依法要求披露的除外。
4. 本条保密义务不因本协议停止或终止而失效。

## 第十三条 对外宣传与案例使用

1. 未经甲方书面同意，乙方不得使用甲方学校名称、校徽、校内截图、老师反馈、学生材料、项目数据或合作信息进行对外宣传、案例展示、媒体发布或商业推广。
2. 如双方后续需要对外展示合作成果，应另行确认展示内容、展示范围、脱敏方式和发布口径。

## 第十四条 违约责任

1. 任一方违反本协议约定，给对方造成实际损失的，应承担相应责任。
2. 乙方因故意或重大过失造成甲方资料泄露、数据丢失或超范围使用的，应依法承担相应责任。
3. 甲方未按约定支付服务费用且经乙方书面催告后仍未支付的，乙方有权暂停相应服务；甲方行使本协议第三条约定的停止服务权不视为违约。

## 第十五条 协议终止与数据处理

1. 甲方按本协议第三条通知停止服务的，本协议项下服务于双方确认的停止日期终止。
2. 双方协商一致，可随时终止本协议。
3. 协议终止后，乙方应按甲方要求配合资料导出、账号关闭、权限撤销和数据删除或停止处理。
4. 协议终止不影响双方就终止前已发生事项进行结算。

## 第十六条 争议解决

1. 本协议适用中华人民共和国法律。
2. 因本协议产生的争议，双方应先友好协商解决；协商不成的，任一方可向甲方所在地有管辖权的人民法院提起诉讼。

## 第十七条 其他

1. 本协议未尽事宜，双方可通过补充协议、项目任务单、会议纪要或其他书面形式另行确认。
2. 本协议一式两份，甲乙双方各执一份，具有同等法律效力。
3. 本协议经双方授权代表签字或盖章后生效。
