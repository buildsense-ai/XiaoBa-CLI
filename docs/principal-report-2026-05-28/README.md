## 明天汇报的核心判断

这次汇报不是普通工具试用复盘，而是要让校长形成一个明确判断：

**第一阶段试点已经证明 AI 可以进入老师真实工作；第二阶段应升级为学校级“广大附教育智能体”；首批聚焦排课、总务台账、学校资料库，并把成果整理成可展示、可推荐、可申报的 AI 教育融合材料。**

## 推进优先级

1. 先定 PPT 主线。
2. 再补 AI 示范校园方案，作为 PPT 后半部分的战略与政策支撑。
3. 再起草服务协议，方便明天直接推进合作。
4. 最后整理演示内容，确保现场可跑，且有截图/录屏兜底。
