---

📌 它到底是什么？

简单说：一个跑在你自己电脑上的私人AI助手。

你在任何聊天软件里给它发消息，它就能回复你。不是那种网页版的ChatGPT套壳，是真正接入你日常用的通讯工具里的AI。

支持的平台多到离谱：WhatsApp、Telegram、Slack、Discord、Signal、iMessage、微软Teams、Google Chat……甚至还有Matrix和WebChat 😱

---

🔥 为什么我觉得它很牛？

1️⃣ 本地优先，数据是你的
不用把聊天记录交给第三方服务器，Gateway跑在你自己机器上，隐私拉满！

2️⃣ 全平台通吃
Mac、iOS、Android、Linux、Windows（WSL2）全都支持。还有macOS菜单栏App和手机端节点，体验很丝滑

3️⃣ 语音也能玩
Voice Wake唤醒 + Talk Mode对话，支持ElevenLabs语音合成，直接跟AI说话，科幻感拉满 🎙️

4️⃣ 不只是聊天
浏览器控制、定时任务、Webhook、Gmail监听、摄像头截图、屏幕录制……工具链超丰富，能帮你自动化很多事情

5️⃣ 模型随便换
推荐用Claude（Anthropic），也支持OpenAI，模型还能自动故障切换，不怕掉线

---

⚡ 上手有多简单？

三行命令搞定：

npm install -g openclaw@latest
openclaw onboard --install-daemon
openclaw gateway --port 18789

然后跟着引导wizard一步步配就行，小白也能搞定～
