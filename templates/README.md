# 论文模板 (Paper Templates)

本目录包含各种学术论文模板，帮助你快速开始论文写作。

## 可用模板

### 1. 研究论文模板 (Research Paper Template)
**文件**: `research-paper-template.md`

适用于实验性研究论文，包含标准的研究论文结构：
- 摘要 (Abstract)
- 引言 (Introduction)
- 相关工作 (Related Work)
- 方法 (Methodology)
- 实验 (Experiments)
- 讨论 (Discussion)
- 结论 (Conclusion)

**适用场景**:
- 提出新方法或算法的论文
- 实验性研究
- 系统设计论文

### 2. 综述论文模板 (Survey Paper Template)
**文件**: `survey-paper-template.md`

适用于综述类论文，包含综述论文的标准结构：
- 摘要 (Abstract)
- 引言 (Introduction)
- 背景知识 (Background)
- 主题分类 (Topic Categories)
- 比较分析 (Comparative Analysis)
- 挑战与机遇 (Challenges and Opportunities)
- 结论 (Conclusion)

**适用场景**:
- 文献综述
- 领域调研
- 技术综述

## 使用方法

```

```

### 方法 2: 手动复制模板

1. 选择合适的模板文件
2. 复制到你的工作目录
3. 根据你的研究内容填充各个章节

## 模板定制

所有模板都可以根据具体需求进行定制：
- 添加或删除章节
- 调整章节顺序
- 修改标题层级
- 适配特定期刊/会议的格式要求

## 格式转换

模板使用 Markdown 格式，可以方便地转换为其他格式：
- Word (.docx)
- PDF (.pdf)
- LaTeX (.tex)

使用 XiaoBa 的 `convert_tool` 可以轻松完成格式转换。
